# Public Source Manifests

This directory publishes the public source manifests used by the repository's provenance model.

Each manifest is a sanitized, publication-safe summary of a bounded public source family that anchors one or more governance artifacts in this repository. These manifests make attribution inspectable without exposing internal research notes or private drafting material, but they are only trustworthy when the manifest scope, the artifact `source_basis`, and the declared `source_manifests` all align.
